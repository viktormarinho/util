cores.py

A library made to facilitate using colors in the console with python

This library its a simple and light-weight archive that you can include in your python project easily. Just leave the file anywhere you want in the project and then use import with the relative file path, and you're good to go! - Recommendation: utilize from cores import *. It will make your life a lot easier when you are using the library.

In essence, the usage of cores.py is by doing string interpolation to put color in the text, using a more comprehensible and descriptive way than the normal '\033[96m' cryptic code. In cores.py, you can simply put cores['cyan'] in your string and it will have the same effect.

There is a way that i, personally, find the best to use the interpolations, and that is following the fstring model:
f"{cores['bgreen']}Hello mom!{clear}"
Okay, let me explain some things here. At first glance, you may think that bgreen is a typo, but no, it actually refers to brighter colors. So, when you want to use a brighter blue, per example, you would use bblue. Secondly, there is another variable being used, the clear variable. The clear variable is a special one, it is used to remove all the colors, backgrounds and effects that the text have at the point that it is used. Why would you do this? Simple. Because of the way that the colors work in the terminal, once you use any style on console text, it will automatically extend to the end of the text, or until it gets overwritten by another style. So, to solve this problem we use the clear variable.

One thing that i need to warn you is, that different terminals have different colors. The Windows prompt won't display the exact same colors as the VS Code terminal, and it won't display the exact same colors as the PyCharm terminal. Every terminal has its own set of colors, they interpret these values as different colors. Some might be identical, some might be a little bit off.

I strongly recommend that you open the file in order to understand how the text effects work, and it is fairly simple. The cores.py module haves a lot of comments in it, explaining everything you need to know. The basic syntax consists of: cores['COLORNAME'] to use colors, bg['COLORNAME'] to use backgrounds and fx['FXNAME'] to use effects like bold or italic. You can use one color, one bg and one fx on the same text at a time.
